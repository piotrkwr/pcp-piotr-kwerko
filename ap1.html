Description du projet :

Mise en place d'un domaine Active Directory complet pour une entreprise fictive. Ce projet simule une gestion de parc informatique centralisée.

Missions réalisées :

  • • Windows Server : Installation et promotion du rôle AD DS.
  • • Gestion GPO : Déploiement de stratégies de sécurité (fond d'écran, lecteurs réseaux).
  • • DNS/DHCP : Configuration des étendues et de la résolution de noms.
